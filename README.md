# Sistema de Cadastro de Alunos em Python

Projeto desenvolvido para praticar lógica de programação e manipulação de dados com Python.

## Funcionalidades

- Cadastro de alunos
- Listagem de alunos
- Busca por nome
- Remoção de alunos
- Persistência de dados com JSON

## Tecnologias utilizadas

- Python
- JSON

## Como executar

1. Clone o repositório:

git clone https://github.com/pfbruno/sistema-alunos-python.git

2. Acesse a pasta:

cd sistema-alunos-python

3. Execute o programa:

python main.py

## Exemplo de uso

1 - Cadastrar aluno  
2 - Listar alunos  
3 - Buscar  
4 - Remover  
5 - Sair
